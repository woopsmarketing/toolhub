## Text 도구 (7개)

### 1. word-counter — 글자수 세기
- **UX**: 텍스트 입력 → 실시간 통계 카드 (8개 지표)
- **출력**: 글자(공백포함/미포함), 단어, 문장, 문단, 줄, 바이트(UTF-8), 읽기 시간
- **사용 시나리오**: 자기소개서 글자수 확인, SNS 글자수 제한 체크, 메타 디스크립션 길이 확인
- **차별점**: 한국어 읽기 시간 계산 (500자/분 기준)

### 2. case-converter — 대소문자 변환기
- **UX**: 텍스트 입력 → 변환된 텍스트 출력
- **지원 변환**: UPPER, lower, Title Case, camelCase, snake_case, kebab-case
- **사용 시나리오**: 코드 변수명 변환, 제목 포맷팅, CSS 클래스명 변환

### 3. duplicate-line-remover — 중복 줄 제거
- **UX**: 텍스트 입력 → 중복 제거된 텍스트 출력
- **사용 시나리오**: 엑셀 데이터 정리, 로그 파일 중복 제거, 이메일 목록 정리

### 4. text-reverser — 텍스트 뒤집기
- **UX**: 텍스트 입력 → 역순 텍스트 출력
- **지원 모드**: 전체 역순, 줄별 역순
- **사용 시나리오**: 암호화 놀이, 팰린드롬 확인, 데이터 변환

### 5. slug-generator — URL 슬러그 생성기
- **UX**: 텍스트 입력 → URL-safe 슬러그 출력
- **변환**: 공백→하이픈, 특수문자 제거, 소문자 변환
- **사용 시나리오**: 블로그 포스트 URL 생성, SEO 친화적 URL 만들기

### 6. text-diff — 텍스트 비교
- **UX**: 텍스트 입력 → 줄 단위 diff 출력 (+/- 표시)
- **사용 시나리오**: 코드 변경사항 확인, 문서 버전 비교, 번역 전후 대조

### 7. lorem-ipsum-generator — 더미 텍스트 생성
- **UX**: 단락 수 입력 → Lorem Ipsum 텍스트 생성
- **템플릿**: FormToResult
- **사용 시나리오**: 웹 디자인 목업, UI 프로토타입, 인쇄물 레이아웃

## Developer 도구 (8개)

### 8. json-formatter — JSON 포맷터
- **UX**: JSON 입력 → 들여쓰기 포맷팅 + 유효성 검증
- **사용 시나리오**: API 응답 가독성 개선, JSON 디버깅, 설정 파일 정리

### 9. base64-encoder — Base64 인코더/디코더
- **UX**: 텍스트 입력 → 자동 감지 후 인코딩 또는 디코딩
- **사용 시나리오**: API 토큰 디코딩, 이미지 Base64 변환, 인증 헤더 확인

### 10. url-encoder — URL 인코더/디코더
- **UX**: URL 입력 → 퍼센트 인코딩/디코딩
- **사용 시나리오**: 한글 URL 파라미터 처리, API 쿼리 스트링 구성, 리다이렉트 URL 디버깅

### 11. html-entity-converter — HTML 엔티티 변환기
- **UX**: 텍스트 입력 → HTML 엔티티 변환 (&amp; &lt; &gt; 등)
- **사용 시나리오**: HTML 소스 내 특수문자 이스케이프, 코드 블로그 포스팅

### 12. jwt-decoder — JWT 디코더
- **UX**: JWT 토큰 입력 → 헤더/페이로드 파싱 출력
- **사용 시나리오**: 인증 토큰 디버깅, 만료 시간 확인, 클레임 값 확인

### 13. unicode-converter — 유니코드 변환기
- **UX**: 텍스트 입력 → \uXXXX 이스케이프 시퀀스 상호 변환
- **사용 시나리오**: 다국어 문자 디버깅, JSON 유니코드 이스케이프 해석

### 14. regex-tester — 정규식 테스터
- **UX**: 패턴 + 테스트 문자열 입력 → 매칭 결과 + 캡처 그룹 출력
- **사용 시나리오**: 정규식 디버깅, 데이터 추출 패턴 테스트, 유효성 검사 규칙 확인

### 15. markdown-preview — 마크다운 미리보기
- **UX**: 마크다운 입력 → HTML 렌더링 출력
- **사용 시나리오**: README 작성, 블로그 글 미리보기, 문서 작성

## Calculator 계산기 (7개)

### 16. percentage-calculator — 퍼센트 계산기
- **UX**: 값 + 퍼센트 입력 → 결과값, 증가 결과, 감소 결과
- **사용 시나리오**: 쇼핑 할인 계산, 세금 계산, 성적 비율 계산

### 17. date-calculator — 날짜 계산기
- **UX**: 시작일 + 종료일 입력 → 일수, 주, 개월 차이
- **사용 시나리오**: D-Day 계산, 프로젝트 기간 산출, 생후 일수 확인

### 18. loan-calculator — 대출 계산기
- **UX**: 대출금 + 금리 + 기간 입력 → 월 상환액, 총 이자, 총 상환액
- **사용 시나리오**: 주택담보대출 비교, 학자금 대출 계획, 자동차 할부 계산

### 19. bmi-calculator — BMI 계산기
- **UX**: 키 + 몸무게 입력 → BMI 수치, 비만도 판정
- **사용 시나리오**: 건강 체크, 다이어트 목표 설정, 건강검진 대비

### 20. age-calculator — 나이 계산기
- **UX**: 생년월일 입력 → 만 나이, 한국 나이, 살아온 일수, 다음 생일
- **차별점**: 한국 나이(세는 나이) 동시 표시
- **사용 시나리오**: 공문서 만 나이 확인, 생일 D-Day, 연령 기준 확인

### 21. discount-calculator — 할인 계산기
- **UX**: 원래가격 + 할인율 입력 → 할인금액, 최종가격
- **사용 시나리오**: 쇼핑 할인가 계산, 세일 비교, 쿠폰 적용 후 가격 확인

### 22. salary-calculator — 연봉 실수령액 계산기
- **UX**: 연봉 + 부양가족 + 식대 + 퇴직금 옵션 → 9개 공제 항목별 상세 결과
- **차별점**: 2026년 최신 4대보험 요율 반영, 소득세 누진세율 적용
- **사용 시나리오**: 이직 시 연봉 비교, 신입 첫 월급 예측, 연봉 인상 효과 확인

## Converter 변환 도구 (2개)

### 23. unit-converter — 단위 변환기
- **UX**: 값 + 변환 전/후 단위 선택 → 변환 결과
- **지원 단위**: km, m, cm, mm, mile, ft, inch 등
- **사용 시나리오**: 레시피 단위 변환, 해외 직구 사이즈 확인, 학업 단위 환산

### 24. color-converter — 색상 코드 변환기
- **UX**: 색상 코드 입력 → HEX, RGB, HSL 상호 변환
- **사용 시나리오**: 웹 디자인 색상 코드 변환, CSS 색상값 변환, 디자인 시스템 작업

## Generator 생성 도구 (2개)

### 25. password-generator — 비밀번호 생성기
- **UX**: 길이 + 문자 유형 선택 → 랜덤 비밀번호 생성
- **옵션**: 대문자, 소문자, 숫자, 특수문자 포함 여부
- **사용 시나리오**: 계정 보안 강화, 비밀번호 정책 충족, 일회용 비밀번호 생성

### 26. uuid-generator — UUID 생성기
- **UX**: 생성 개수 선택 → UUID v4 목록 생성
- **사용 시나리오**: 데이터베이스 PK 생성, API 요청 ID 생성, 테스트 데이터 생성
