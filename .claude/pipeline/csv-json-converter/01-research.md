## 검색 의도
- **주요 의도 (90%)**: 즉시 사용 가능한 무료 온라인 변환 도구. 로그인/설치 없이 텍스트 붙여넣기 → 결과 복사
- **보조 의도 (10%)**: 변환 원리, 중첩 JSON 처리, Python/jq 같은 코드 대안
- **페인 포인트**:
  - 파일 업로드 강제에 대한 거부감 (텍스트 붙여넣기 선호)
  - 서버 업로드로 인한 데이터 유출 우려 → "브라우저 로컬 처리" 수요 강함
  - 중첩 JSON을 CSV로 변환할 때의 평면화 문제
  - 헤더 자동 감지, 따옴표/쉼표 이스케이프, 한글 UTF-8 BOM 문제

## 경쟁 현황
- **csvjson.com**: "데이터가 서버로 전송되지 않음" 명시, 양방향 지원, 영문 전용
- **convertcsv.com**: 5가지 JSON 출력 모드, 파워유저 타겟, UI 복잡
- **jsonformatter.org**: SEO 강력, 디자인 낡음, 광고 많음
- **jam.dev/utilities/csv-to-json**: 광고 없음, 깔끔한 UI (벤치마크)
- **site24x7.com/ko**: 한국어 UI 제공, 기능 단순
- **차별화 포인트**:
  1. 한국어 UI + 양방향 자동 감지 조합이 희박함
  2. 붙여넣기 입력 + 로컬 처리 강조
  3. 한글 UTF-8 BOM 옵션, 구분자(,/;/탭) 선택
  4. FAQ 기반 SEO 콘텐츠 강화 여지 큼
